ReplayPro v32.1 Stable

This is a stable resend based on the known-good v30 build.

Included:
- checkbox playlist in Clip / Media
- PLAY SELECTED plays checked clips in order
- HOLD on last frame after replay
- TAKE LIVE returns to live

Reason for this build:
- the experimental v32 HDMI pipeline changes caused compile errors
- this package rolls back to the stable, building project so it can open and build cleanly in Visual Studio
